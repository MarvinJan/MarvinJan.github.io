Cubify

Cubify - a JavaScript plugin

Introduction

A JavaScrip plugin that allows you to transform up to 6 of your DOM blocks into an interactive 3d Cube!

For example! Here we have 6 blocks we wish to make a cube:

FRONT-FACE BLOCK
BACK-FACE BLOCK
TOP-FACE BLOCK
BOTTOM-FACE BLOCK
LEFT-FACE BLOCK
RIGHT-FACE BLOCK

Now, all we have to do is add cubify script into our head tag:

<script src="YOUR__PATH/cubify.js"></script>

Then, initiate a new Cubify instance, passing those blocks in the constructor like this: 

const front =  document.querySelector(".example-block__front");
const back =  document.querySelector(".example-block__back"); 
const top = document.querySelector(".example-block__top"); 
const bottom = document.querySelector(".example-block__bottom"); 
const left = document.querySelector(".example-block__left"); 
const right = document.querySelector(".example-block__right"); 

const exampleCube = new Cubify( front, /* front face as first param */
{ /* options object */ 
  faces: {
    back: back,
    top: top,
    bottom: bottom,
    left: left,
    right: right
  },
  interactive: true
})

And the result we get is:

FRONT-FACE BLOCK
BACK-FACE BLOCK
TOP-FACE BLOCK
BOTTOM-FACE BLOCK
LEFT-FACE BLOCK
RIGHT-FACE BLOCK

Options

Options are set inside the second parameter of the Cubify class constructor function: 

const exampleCube = new Cubify( firstParameter /* front face of the cube */, 
{ /* options object */ })

Here are the available options:

faces : object

With faces you can pass the needed elements as cube's 5 remainind faces, like that: 

{ /* options object */ 
    faces{
      back: /* YOUR__ELEMENT */,
      top: /* YOUR__ELEMENT */,
      bottom: /* YOUR__ELEMENT */,
      left: /* YOUR__ELEMENT */,
      right: /* YOUR__ELEMENT */
    }
})

As evident, has 5 keys: top, bottom, left, right, back

interactive : boolean (default : false)

An option to allow user to rotate the cube with mouse.

width : number (default : - largest dimension (meaning height or width) of all passed DIVs)

The width, height and depth of the cube in pixels.
